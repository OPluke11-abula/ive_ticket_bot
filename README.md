# IVE Tixcraft Userscript

這個資料夾目前主要提供一支可放進 Tampermonkey 的拓元輔助腳本：

- [`itzy_ticket_bot.user.js`](D:\GitHub\ive_ticket_bot\itzy_ticket_bot.user.js)

雖然檔名還叫 `itzy_ticket_bot.user.js`，但內容已經改成 IVE 正式版預設。

## 目前正式版預設

- 活動頁：`https://tixcraft.com/activity/detail/26_ive`
- 目標場次：`2026/09/12 18:00`
- 開始啟動時間：`2026/04/03 10:59:58`
- 開賣資訊：`2026/04/03 (五) 11:00 全面開賣`
- 張數：`2`
- 優先價位：`5800 -> 4800 -> 3800 -> 2800 -> 2300 -> 800`

## 腳本會做的事

1. 在開始時間以前只顯示倒數，不自動點擊、不自動重整。
2. 到 `2026/04/03 10:59:58` 會開始進入檢查流程。
3. 如果停在活動頁，會保守地重整頁面並找「立即購票」。
4. 進到場次頁後，會找 `2026/09/12 18:00` 這場，並等待「立即訂購」可按。
5. 進到區域頁後，依優先價位順序挑票。
6. 進到訂單頁後，會幫忙選張數與勾同意條款。
7. 遇到驗證碼或風控頁時會停手，留給人工處理。

## 目前節奏

這版刻意不用高頻刷新，避免頁面自己先不穩：

- 檢查間隔：`1000 ms`
- 重整冷卻：`1500 ms`
- 點擊冷卻：`600 ms`

## 使用方式

1. 安裝 Tampermonkey。
2. 建立新腳本。
3. 把 [`itzy_ticket_bot.user.js`](D:\GitHub\ive_ticket_bot\itzy_ticket_bot.user.js) 的內容整份貼進去並儲存。
4. 開啟拓元頁面並確認已登入。
5. 右上角面板可以看到目前狀態、倒數、設定與上次動作。

## 面板可調整項目

- 活動網址
- 目標日期
- 目標時間
- 開始時間
- 張數
- 檢查間隔
- 區域優先清單

開始時間欄位支援到秒，格式是瀏覽器的 `datetime-local`，例如：

- `2026-04-03T10:59:58`

## 注意

- 這支腳本是輔助測試與流程學習用途。
- 驗證碼仍然要人工輸入。
- 如果你之前在 Tampermonkey 裡跑過舊版設定，這版已改用新的 localStorage key，不會直接吃到舊設定。
