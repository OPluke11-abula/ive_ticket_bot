# IVE Tixcraft 拓元搶票機器人操作手冊

這是一套專為 IVE WORLD TOUR ＜SHOW WHAT I AM＞ IN TAIPEI 打造的半自動化搶票腳本。本腳本設計核心為**「追求極速進場與鎖票」**，並把最終的驗證碼輸入留給人類處裡，以避免被系統防爬蟲機制阻擋。

---

## 🚀 核心功能與流程

腳本執行後會自動完成以下動作：
1. **進入售票主頁**：直接跳轉 IVE 的售票介面。
2. **監聽開賣狀態**：不斷重整畫面，直到找到指定場次 (預設: `2026/09/12 18:00`) 並出現「立即訂購」按鈕。
3. **強制電腦配位**：進入票區選擇時自動點選「電腦配位」，省略選位拉扯時間。
4. **搶奪高價區**：從 $3,800 開始往下依序尋找 $2,800、$2,300、$800、$400 的票價，一有空位瞬間點入。
5. **秒選票數與條款**：進入結帳首頁後，直接幫您選擇數量 (預設 `2` 張) 並打勾「我同意本站購票須知」。

> **最終階段，人類出馬：** 畫面將停留在打字區，**您只需快速輸入四碼英文字母驗證碼，然後手動按下「確認張數」！**

---

## 🛠️ 事前準備與注意事項 (非常重要)

### 1. 確保登入狀態與清空購物車
- 在執行腳本前，**強烈建議您提早 30 分鐘開啟 Chrome 瀏覽器並登入好拓元帳號**。
- 請確認您的帳號已經做過「手機驗證」，否則按了購買也會被阻擋。
- 此腳本會開啟並操控 Chrome，它會帶入您預設的無痕/乾淨狀態，因此程式「啟動後」它會有一段時間顯示首頁，**請務必在開賣前手動在這個被它打開的瀏覽器視窗內成功登入！**

### 2. 關於驗證碼
- **機器人不負責打驗證碼**。任何號稱能自動打驗證碼的腳本都有極高機率被拓元判定為機器人並封鎖。
- 請確保雙手已經在鍵盤上，當畫面停留在輸入框時，**不要猶豫，半秒內看完驗證碼打出來按 Enter / 點選提交**。

### 3. 如何更改搶票張數？
若您想自己一個人看，或者幫 4 個朋友買，請用記事本或 IDE 打開 `ive_ticket_bot.py`，更改以下設定：
```python
TICKET_QTY = 2  # 改成您要的張數 (例如: 1, 3, 4)
```
> 若拓元該區每人限購張數小於您的設定值，程式會自動抓最大可買張數 (例如限購 2，您設定 4，程式只會選到 2)。

### 4. 如何啟動腳本？
1. 打開您的終端機 (Terminal / PowerShell)。
2. 將路徑切換至您的腳本資料夾：
   ```powershell
   cd D:\GitHub\ive_ticket_bot
   ```
3. 執行指令：
   ```powershell
   python ive_ticket_bot.py
   ```
4. 看到出現 `🚀 開始進入拓元售票系統...` 的提示後，您就可以在跳出的 Chrome 視窗內先進行登入。程式在開賣前會不斷幫您自動重新整理。

## ⚠️ 常見錯誤與除錯

- **找不到 ChromeDriver**：這代表您的電腦沒有安裝 `selenium` 套件。
  - 解法：在終端機輸入 `pip install selenium` 安裝。
- **重新整理過快被伺服器阻擋**：目前的設定是一秒重整一次，若還是被拓元擋下出現 403 錯誤，可在 `ive_ticket_bot.py` 中適當增加 `time.sleep(1)` 的秒數。
